# 构建前端说明

## ✅ 已创建的文件

我已经创建了以下文件：
1. `index.html` - HTML 入口文件
2. `vite.config.ts` - Vite 配置文件
3. `src/main.tsx` - React 入口文件（TypeScript）
4. `构建前端.bat` - 构建脚本

## 🚀 构建步骤

### 方法一：使用批处理文件（推荐）

双击运行 `构建前端.bat` 文件。

### 方法二：手动构建

在命令行执行：

```bash
npm run build
```

## 📝 如果构建失败

如果遇到 TypeScript 错误，可能需要创建 `tsconfig.json`。告诉我错误信息，我会帮你创建。

## 🎯 构建成功后

构建成功后，会生成 `dist` 文件夹，包含：
- `index.html`
- `assets/` 文件夹（包含 JS、CSS 等静态资源）

## 📤 部署到服务器

构建成功后，在服务器上执行：

```bash
cd /www/wwwroot/feihub

# 拉取最新代码
git pull origin main

# 如果服务器上有 Node.js，重新构建前端
npm run build

# 或者直接使用本地构建的 dist 文件夹（需要上传到服务器）
```
