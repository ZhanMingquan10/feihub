# 宝塔面板套件安装指南

## 🎯 选择建议

### 推荐：选择 **LNMP** 套件

**原因：**
- ✅ 包含 Nginx（Web服务器，必须）
- ✅ MySQL 可以保留（虽然我们不用，但不影响）
- ✅ PHP 可以不用管（不影响我们的项目）
- ✅ 安装速度快，一键完成

---

## 📦 安装步骤

### 第一步：安装 LNMP 套件

1. **在宝塔面板中，点击"一键安装"或"安装套件"**
2. **选择"LNMP"**
3. **配置版本（推荐）：**
   - **Nginx**：选择最新稳定版（如 1.24.x 或 1.25.x）
   - **MySQL**：可以选择 8.0（虽然我们不用，但可以保留）
   - **PHP**：选择任意版本（我们不用，但可以保留，不影响）
   - **phpMyAdmin**：可选，我们不用
4. **点击"一键安装"**
5. **等待安装完成**（可能需要 10-30 分钟）

### 第二步：安装我们需要的软件

LNMP 安装完成后，我们需要额外安装：

#### 1. 安装 Node.js
1. 点击左侧"软件商店"
2. 搜索"Node.js版本管理器"
3. 点击"安装"
4. 安装完成后，点击"设置"
5. 选择 Node.js **18.x** 或 **20.x** 版本
6. 点击"安装"
7. 等待安装完成

#### 2. 安装 PostgreSQL
1. 在"软件商店"搜索"PostgreSQL"
2. 点击"安装"，选择最新版本（建议 15.x 或 16.x）
3. 等待安装完成

#### 3. 安装 Redis
1. 在"软件商店"搜索"Redis"
2. 点击"安装"，选择最新版本
3. 等待安装完成

#### 4. 安装 PM2（进程管理器）
1. 点击左侧"终端"
2. 执行：
   ```bash
   npm install -g pm2
   ```

---

## ✅ 安装完成后的检查

在宝塔终端执行以下命令，验证安装：

```bash
# 检查 Node.js
node -v
npm -v

# 检查 Nginx
nginx -v

# 检查 PostgreSQL
psql --version

# 检查 Redis
redis-cli --version

# 检查 PM2
pm2 -v
```

---

## 📝 安装清单

安装完成后，你应该有：
- ✅ Nginx（Web服务器）
- ✅ Node.js 18+（运行后端）
- ✅ PostgreSQL（数据库）
- ✅ Redis（缓存）
- ✅ PM2（进程管理器）
- ⚠️ MySQL（可以保留，我们不用）
- ⚠️ PHP（可以保留，我们不用）
