# 构建前端并部署

## 🏗️ 第一步：构建前端

在命令行执行：

```bash
# 直接使用 Vite 构建（跳过 tsc）
npx vite build
```

如果成功，会看到类似这样的输出：
```
✓ built in 5.23s
dist/index.html                   0.45 kB
dist/assets/index-xxx.js          xxx kB
```

---

## ✅ 第二步：验证构建结果

```bash
# 检查 dist 文件夹
dir dist
```

如果看到 `dist` 文件夹，说明构建成功。

---

## 🚀 第三步：在服务器上拉取并部署

在宝塔终端执行：

```bash
cd /www/wwwroot/feihub

# 拉取最新代码
git pull origin main

# 如果服务器上有 Node.js，重新构建前端
npm run build

# 或者直接使用本地构建的 dist 文件夹（需要上传到服务器）
```

---

## 📝 如果 vite build 失败

如果 `npx vite build` 也失败，可能需要创建 `tsconfig.json`。告诉我错误信息，我会帮你创建。
