# 宝塔面板添加网站指南

## 📍 添加网站的位置

在宝塔面板：
1. 点击左侧 **"网站"** 菜单
2. 在页面顶部，选择 **"PHP项目"** 页签
3. 点击 **"添加站点"** 按钮

---

## 🔧 详细步骤

### 步骤 1：选择页签

在宝塔面板的"网站"页面，选择 **"PHP项目"** 页签（即使前端是静态文件，也选择这个）。

---

### 步骤 2：添加站点

1. 点击 **"添加站点"** 按钮
2. 填写信息：
   - **域名**：`feihub.top`（如果要支持 www，可以填写 `feihub.top,www.feihub.top`）
   - **备注**：`FeiHub 飞书文档知识分享平台`（可选）
   - **根目录**：`/www/wwwroot/feihub/dist`
   - **FTP**：不创建（不需要）
   - **数据库**：不创建（不需要）
   - **PHP版本**：纯静态（重要！选择这个）
3. 点击 **"提交"**

---

## ⚠️ 重要提示

- **PHP版本**：必须选择 **"纯静态"**，因为前端是 React 构建的静态文件
- **根目录**：确保指向 `/www/wwwroot/feihub/dist`（前端构建后的目录）
- **数据库**：不需要创建数据库（后端已经有自己的数据库）

---

## 📝 如果找不到"纯静态"选项

如果 PHP 版本下拉菜单中没有"纯静态"选项：

1. 选择任意 PHP 版本（如 PHP 7.4）
2. 创建网站后，点击站点右侧 **"设置"**
3. 在 **"网站目录"** 中，将 **"运行目录"** 设置为 `/`（根目录）
4. 在 **"默认文档"** 中，确保 `index.html` 在列表中

---

## 🎯 现在执行

1. 在宝塔面板，点击左侧 **"网站"**
2. 选择 **"PHP项目"** 页签
3. 点击 **"添加站点"**
4. 填写信息（记得选择 **"纯静态"**）
5. 点击 **"提交"**
