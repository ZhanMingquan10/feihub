# 配置域名 DNS 解析

## 🔍 问题说明

`ping feihub.top` 显示"找不到主机"，说明域名 DNS 解析还没有配置或未生效。

---

## 🔧 解决方案

### 第一步：在阿里云配置域名解析

1. **登录阿里云控制台**
   - 访问：https://ecs.console.aliyun.com/

2. **进入域名管理**
   - 点击顶部 **"产品与服务"** → **"域名"**
   - 或者直接访问：https://dc.console.aliyun.com/

3. **找到域名 `feihub.top`**
   - 在域名列表中，找到 `feihub.top`
   - 点击右侧 **"解析"** 按钮

4. **添加 A 记录**
   - 点击 **"添加记录"** 按钮
   - 填写：
     - **记录类型**：`A`
     - **主机记录**：`@`（表示 feihub.top）
     - **记录值**：你的服务器公网 IP（在宝塔终端执行 `curl ifconfig.me` 获取）
     - **TTL**：`10分钟`（默认）
   - 点击 **"确认"**

5. **添加 www 记录（可选）**
   - 如果也要支持 `www.feihub.top`，再添加一条：
     - **记录类型**：`A`
     - **主机记录**：`www`
     - **记录值**：你的服务器公网 IP
     - **TTL**：`10分钟`
   - 点击 **"确认"**

---

### 第二步：获取服务器公网 IP

在宝塔终端执行：

```bash
curl ifconfig.me
```

**记录这个 IP，用于配置 DNS。**

---

### 第三步：等待 DNS 生效

DNS 解析生效需要时间：
- **最快**：几分钟
- **通常**：10-30 分钟
- **最长**：24 小时（全球 DNS 传播）

---

## 🚀 临时测试方案

在 DNS 生效之前，可以先用服务器 IP 测试：

### 方法一：直接使用 IP 访问

1. **获取服务器 IP**：
   ```bash
   curl ifconfig.me
   ```

2. **在浏览器访问**：`http://你的服务器IP`
   - 如果可以看到网站，说明配置完全正确
   - 只是等待 DNS 生效即可

---

### 方法二：修改本地 hosts 文件（临时）

在本地电脑（Windows）：

1. **打开 hosts 文件**
   - 路径：`C:\Windows\System32\drivers\etc\hosts`
   - 用管理员权限打开记事本，然后打开这个文件

2. **添加记录**
   ```
   你的服务器IP feihub.top
   你的服务器IP www.feihub.top
   ```

3. **保存文件**

4. **测试访问**
   - 在浏览器访问：`http://feihub.top`
   - 应该可以看到网站

**注意**：这只是临时方案，DNS 生效后可以删除 hosts 文件中的记录。

---

## 📋 配置步骤总结

1. **获取服务器 IP**（在宝塔终端）：
   ```bash
   curl ifconfig.me
   ```

2. **在阿里云配置 DNS**：
   - 域名 → `feihub.top` → 解析 → 添加记录
   - 记录类型：A
   - 主机记录：@
   - 记录值：服务器 IP

3. **等待 DNS 生效**（10-30 分钟）

4. **验证**：
   - 在本地电脑执行 `ping feihub.top`
   - 应该能 ping 通，返回服务器 IP
