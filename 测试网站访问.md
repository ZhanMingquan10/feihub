# 测试网站访问

## ✅ 配置已更新

- Nginx 配置已修改
- Nginx 已重载
- 警告信息不影响功能（只是提示有多个默认 server）

---

## 🚀 现在测试

### 第一步：在浏览器测试访问

1. **获取服务器 IP**（如果还不知道）：
   ```bash
   curl ifconfig.me
   ```

2. **在浏览器访问**：`http://你的服务器IP`
   - 应该能看到 FeiHub 网站首页
   - 如果还是显示"没有找到站点"，继续下一步排查

---

### 第二步：如果还是无法访问

在宝塔终端执行：

```bash
# 1. 检查配置文件是否已修改
grep "server_name" /www/server/panel/vhost/nginx/feihub.top.conf

# 2. 测试本地访问
curl -H "Host: feihub.top" http://localhost | head -20

# 3. 检查是否有其他默认站点
grep -r "server_name.*_" /www/server/panel/vhost/nginx/*.conf

# 4. 查看 Nginx 访问日志
tail -20 /www/wwwlogs/feihub.top.log
```

---

## 🔧 如果还是显示"没有找到站点"

### 方案一：检查默认站点

可能是有其他站点配置了默认 server，覆盖了我们的配置。

```bash
# 查看所有站点配置
ls -la /www/server/panel/vhost/nginx/*.conf

# 检查哪个站点是默认的
grep -l "default_server" /www/server/panel/vhost/nginx/*.conf
```

---

### 方案二：在宝塔面板检查站点列表

1. 在宝塔面板，点击左侧 **"网站"**
2. 查看站点列表
3. 检查是否有其他站点也配置了默认 server
4. 如果有，可以临时禁用或删除

---

### 方案三：直接修改配置文件（更彻底）

在宝塔文件管理器中：
1. 进入 `/www/server/panel/vhost/nginx/`
2. 编辑 `feihub.top.conf`
3. 在 `server_name` 行，确保是：
   ```nginx
   server_name feihub.top www.feihub.top _;
   ```
4. 保存文件
5. 在宝塔面板重载 Nginx

---
