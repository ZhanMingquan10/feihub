# 查看完整日期提取日志

## 🔍 查看详细的日期提取日志

在宝塔终端执行以下命令，并把**完整的输出**发给我：

```bash
cd /www/wwwroot/feihub/backend

# 1. 查看最近的文档处理日志（包含日期提取信息）
pm2 logs feihub-backend --lines 500 --nostream | grep -E "(Puppeteer|日期|date|提取|处理文档|爬取)" | tail -100

# 2. 或者查看所有输出日志（最近100行）
pm2 logs feihub-backend --lines 100 --nostream

# 3. 特别关注日期相关的日志
pm2 logs feihub-backend --lines 500 --nostream | grep -i "date\|日期\|月\|日" | tail -50
```

---

## 📝 重点关注

请把包含以下关键词的日志发给我：
- `[Puppeteer] 提取的日期`
- `[Puppeteer] 最终提取的日期`
- `[爬取] 从 cheerio 提取的原始日期文本`
- `[爬取] 从 cheerio 提取到日期`
- `[爬取] 最终日期`
- `[处理文档] - 日期`
- `doc-info-time-item`

---

## 🔧 如果看不到日期相关日志

如果日志中没有日期相关的信息，可能是：
1. 代码还没有部署
2. 日志被清空了
3. 需要重新提交一个文档来触发日志

请重新提交一个飞书文档，然后立即查看日志。
