### 第二步：配置域名 DNS 解析（如果还没配置）

在阿里云控制台：

1. **进入域名管理**
   - 访问：https://dc.console.aliyun.com/
   - 找到 `feihub.top` 域名
   - 点击 **"解析"** 按钮

2. **添加 A 记录**
   - 点击 **"添加记录"**
   - 填写：
     - **记录类型**：`A`
     - **主机记录**：`@`（表示 feihub.top）
     - **记录值**：你的服务器公网 IP
     - **TTL**：`10分钟`
   - 点击 **"确认"**

3. **添加 www 记录（可选）**
   - 如果也要支持 `www.feihub.top`，再添加一条：
     - **记录类型**：`A`
     - **主机记录**：`www`
     - **记录值**：你的服务器公网 IP
   - 点击 **"确认"**

4. **等待 DNS 生效**
   - 通常需要 10-30 分钟
   - 可以通过 https://www.whatsmydns.net/ 检查

---

### 第三步：申请 SSL 证书

DNS 生效后，在宝塔面板：

1. **打开站点设置**
   - 网站 → `feihub.top` → 设置

2. **申请 SSL 证书**
   - 点击 **"SSL"** 标签
   - 选择 **"Let's Encrypt"** 标签
   - 勾选域名：`feihub.top`（如果支持 www，也勾选）
   - 点击 **"申请"** 按钮
   - 等待申请完成（可能需要几分钟）

3. **启用 HTTPS**
   - 申请成功后，点击 **"强制HTTPS"** 按钮
   - 这样所有 HTTP 请求会自动跳转到 HTTPS

---

## 🆘 如果遇到问题

### 问题 1：DNS 解析未生效

- 检查阿里云 DNS 配置是否正确
- 等待 DNS 生效（可能需要 10-30 分钟）
- 可以通过 https://www.whatsmydns.net/ 检查

### 问题 2：SSL 证书申请失败

- 检查域名解析是否已生效
- 检查 80 端口是否开放
- 等待几分钟后重试

### 问题 3：功能测试失败

- 检查后端服务是否运行：`pm2 status`
- 检查后端日志：`pm2 logs feihub-backend`
- 检查浏览器控制台是否有错误
