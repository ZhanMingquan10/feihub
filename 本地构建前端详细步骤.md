# 本地构建前端详细步骤

## 📋 第一步：打开命令行

1. 按 `Win + R`，输入 `cmd`，按回车
2. 或者按 `Win + X`，选择"Windows PowerShell"或"终端"

---

## 📂 第二步：进入项目目录

在命令行中输入：

```bash
cd "D:\AI+CPS(只做一件事_做好一件事)\13.出海工具站\feihub"
```

按回车执行。

---

## 🔍 第三步：检查 Node.js 是否安装

输入：

```bash
node --version
npm --version
```

如果显示版本号，说明已安装。如果没有，需要先安装 Node.js。

---

## 📦 第四步：安装依赖（如果还没安装）

输入：

```bash
npm install
```

等待安装完成（可能需要几分钟）。

---

## 🏗️ 第五步：构建前端

输入：

```bash
npm run build
```

等待构建完成。如果成功，会看到类似这样的输出：
```
✓ built in 5.23s
```

构建完成后，会在项目根目录生成 `dist` 文件夹，里面是构建好的前端文件。

---

## ✅ 第六步：验证构建结果

构建完成后，检查 `dist` 文件夹是否存在：

```bash
dir dist
```

如果看到 `dist` 文件夹，说明构建成功。

---

## 🚀 第七步：推送到 GitHub

```bash
# 添加更改
git add .

# 提交
git commit -m "优化正文显示：统一显示10行"

# 推送
git push origin main
```

---

## 📝 如果遇到错误

### 错误1：找不到 node 或 npm
**解决**：需要安装 Node.js
- 访问 https://nodejs.org/
- 下载并安装 LTS 版本

### 错误2：npm install 失败
**解决**：尝试清除缓存后重新安装
```bash
npm cache clean --force
npm install
```

### 错误3：构建失败
**解决**：查看错误信息，通常是 TypeScript 类型错误，需要修复后再构建。

---

完成这些步骤后，前端就构建好了，然后推送到 GitHub，再在服务器上部署。
